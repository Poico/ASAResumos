
## Complexity

```python
r = n;
i = 0;
while(r > 0){
	i++;
	r-=i;
}
```

| r | i | Iteration |
| ---- | ---- | ---- |
| r-1 | 1 | 0 |
| r-1-2 | 2 | 1 |
| r-1-2-3 | 3 | 2 |
| r-1-2-3-4 | 4 | 3 |
| r-1-2-3-4-5 | 5 | 4 |
| r-1-2-3-4-5-6 | 6 | 5 |
| $$ r - i *(i+1)/2 >= n$$ | Iteration+1 | r>0 |
|  |  |  |
